# 🧊 Holo Design System Template

A premium, high-performance Next.js template based on the **Holo Design System**. Built for engineering labs, research portals, and technical portfolios.

## 🚀 Features

- **Framework**: Next.js 16 (App Router) + TypeScript.
- **Styling**: Pure CSS Design Tokens (Vanilla CSS) for maximum control and performance.
- **Typography**: Optimized integration of `Syne`, `Space Grotesk`, and `JetBrains Mono`.
- **UI Components**: Reusable Glassmorphism components (`HoloCard`, `HoloButton`).
- **Research Visuals**: SVG-based animated components (`RobotViz`, `TimelineSpark`).
- **i18n**: Built-in multi-language support (ES/EN) via Context API.
- **Design**: Ambient gradient blobs, subtle grid backgrounds, and smooth anchor navigation.

## 🛠️ Getting Started

1. **Clone & Install**:
   ```bash
   npm install
   ```

2. **Run Development**:
   ```bash
   npm run dev
   ```

3. **Configure Content**:
   Edit `src/constants/content.ts` to customize your data.

## 🎨 Design Tokens

Defined in `src/app/globals.css`:
- `--cya`: Primary Cyan (#22d3ee)
- `--lav`: Primary Lavender (#a5b4fc)
- `--pur`: Deep Purple (#7c3aed)
- `--bg`: Obsidian Black (#08090f)
